03 - JSON

JSON is a format used to store and transfer information. As the name suggests, it is "based" on JavaScript objects, although it is language-independent. JSON files can used to hold structured data, for instance a list of people, or details about an event. Feel free to read Wikipedia entry for JSON for some info and notes.

Example

From JavaScript's point of view, a JSON is nothing but an object (or array). This is an example of JSON showing details about an event: 

{
	id: "alrncxh91jfo",
	name: "Spring Festival",
	date: "2017-03-21 21:00:00",
	place: "Town square"
	description: "Great celebration for this year's spring!",
	note: "Free entrace"
}
As any JavaScript object, it contains pairs in the form key: value. Here is another example with a list of people: 
[
	{
		name: "John",
		surname: "Smith",
		address: "5, Fake Street"
	},
	{
		name: "Pablo",
		surname: "Guerra",
		address: "7, Main Street"
	},
	{
		name: "Daisy",
		surname: "Duck",
		address: "6, Duckbourg"
	}
]

JSON files

A JavaScript object can be safely embedded in a JavaScript piece of code. It is also possible to have a separate JSON file, containing only the JSON object (no code!). Such a file has .json extension, for instance data.json.
